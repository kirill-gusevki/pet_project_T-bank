## О проекте

Этот репозиторий содержит данные и код для анализа поведения пользователей сервиса краткосрочной аренды электросамокатов Т-Банк. Проект посвящён разведочному анализу данных (EDA), формированию профиля пользователей, выявлению ключевых инсайтов и разработке продуктовых гипотез на основе данных поездок.

## Аналитическая записка

### Профиль пользователей

**Демография:**
- **Пол:** Мужчины - 85.2%, Женщины - 14.8% (потенциал роста)
- **Возраст:** Основная аудитория 20-35 лет (средний возраст ~31.5 лет)
- **Семейный статус:** 19% пользователей женаты/замужем - перспективная группа с высокой платежеспособностью

### Поведенческие инсайты

**Характеристики поездок:**
- **Длительность:** Средняя - 11 мин, Медиана - 8 мин (75% поездок < 20 мин)
- **Стоимость:** Средняя - 127.8 руб, Медиана - 104 руб
- **Расстояние:** Среднее - 3.7 км, Медиана - 1.7 км

**Временные паттерны:**
- **Сезонность:** Пик активности - июль-сентябрь
- **Время суток:** Пики с 14:00-17:00 и 4:00-6:00

### Географическое распределение

**ТОП-3 региона:**
1. Москва
2. Санкт-Петербург  
3. Московская область

**Точка роста:** Ростовская область занимает 10 место по поездкам при 6 месте по населению

### Анализ модельного ряда

**Самые популярные модели:**
1. SL
2. Max  
3. E
4. Urban E-Bike

**Самые выгодные модели** (популярность + стоимость):
1. SL
2. Max
3. E
4. Urban E-Bike

**Особенная модель:** Techno - 25% пользователей женщины, 7 место по выгодности

## Продуктовые гипотезы

### Гипотеза №1: Региональная экспансия в Ростовской области
- **Суть:** Фиксированный тариф 100 руб/час + кэшбэк 15% у партнерских локаций
- **Ожидаемый эффект:** Рост доли рынка региона относительно прочих субъектов с 3% до 8%
- **Метрики:** Количество поездок в регионе, активации у партнеров

### Гипотеза №2: Удвоенный кэшбэк для женщин 20-50 лет на модель Techno  
- **Суть:** Удвоение кэшбэка для целевой аудитории на специфичную модель
- **Ожидаемый эффект:** Рост доли женщин с 20% до 35%
- **Метрики:** Доля женщин среди пользователей Techno

### Гипотеза №3: Семейный тариф "Вместе веселее"
- **Суть:** Прогрессивные скидки при аренде нескольких самокатов
- **Ожидаемый эффект:** Рост частоты поездок пар на 40%
- **Метрики:** Количество активаций тарифа

## Рекомендация

**Самая перспективная гипотеза:** **Гипотеза №1 - Региональная экспансия**

**Обоснование:**
- Неосвоенный рынок с высоким потенциалом
